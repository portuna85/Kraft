# Security Policy

## Supported Versions

The `main` branch is the only actively maintained branch.  
Older branches and tags are not patched.

## Reporting a Vulnerability

보안 취약점은 **공개 GitHub Issue로 제보하지 마세요.**

이메일로 비공개 제보해 주세요: **portuna85@gmail.com**

제보 시 다음 내용을 포함해 주시면 빠른 검토에 도움이 됩니다.

- 취약점 유형 (예: XSS, SQL Injection, 인증 우회 등)
- 재현 방법 또는 PoC
- 영향 범위 (데이터, 기능, 사용자 범위)

접수 후 **3 영업일** 이내에 확인 메일을 드립니다.  
패치가 완료된 이후 취약점 내용을 공개할 수 있으며, 제보자를 명시하겠습니다 (원하는 경우).
